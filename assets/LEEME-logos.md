# Logos de los grupos organizadores

El formulario (`index.html`) muestra una fila de logos al final, en la sección
**"Organizan"**. Cada logo se carga desde esta carpeta. Si el archivo no existe,
simplemente no se muestra (no rompe la página).

## Archivos que debes poner aquí

Guarda las 4 imágenes con **exactamente** estos nombres (en minúsculas):

| Archivo                      | Logo                          |
|------------------------------|-------------------------------|
| `maranata-class.png`         | Maranata Class                |
| `escuela-sabatica.png`       | Escuela Sabática              |
| `gteen.png`                  | Generación Teen (Gteen)       |
| `ja.png`                     | Juventud Adventista (JA)      |

## Recomendaciones

- Formato **PNG con fondo transparente** se ve mejor (también sirve JPG).
- Alto sugerido: ~120–200 px (se muestran a 42 px de alto, escalados).
- Pesos livianos (< 150 KB c/u) para que el formulario cargue rápido.

> Si prefieres otros nombres o más/menos logos, edítalos en `index.html`,
> dentro del bloque `<div class="logos">`.
